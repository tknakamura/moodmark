## 🔄 両サイトの相乗効果

### カスタマージャーニー

```
1. 情報収集フェーズ
   ↓
   MOO:D MARK GIFT（SEOメディア）
   • 検索エンジンから流入（97.1%）
   • ギフトの選び方を学ぶ
   • 商品アイデアを得る
   • 週間210,917セッション
   ↓
2. 商品検討フェーズ
   ↓
   MOO:D MARK（ECサイト）
   • SEOメディアから遷移
   • または直接検索で流入
   • 商品を比較・検討
   • 週間136,892セッション
   ↓
3. 購入フェーズ
   ↓
   MOO:D MARK（ECサイト）
   • カートに追加
   • 購入完了（実際の購入CVRは約0.5〜1.0%）
```

### 🎯 シナジー効果の最大化

#### 現在の課題
1. **SEOメディア→ECサイトの導線が不明確**
   - 記事からECサイトへの誘導測定が必要
   - どれくらいのユーザーが遷移しているか不明

2. **クロスサイトのユーザー追跡**
   - 同一ユーザーの行動が追えていない
   - GA4のクロスドメイントラッキング設定が必要

#### 改善施策

**🔴 優先度：高**

1. **SEOメディアからECサイトへの導線強化**
   ```
   施策例：
   • 記事末尾に「ご紹介した商品はこちらから購入できます」
   • 商品カードの挿入（画像・価格・リンク）
   • カテゴリ別のECサイトリンク
   • 各記事に3〜5箇所のEC誘導リンク
   ```

2. **クロスドメイントラッキングの実装**
   ```
   目的：
   • SEOメディア→ECサイトの遷移を測定
   • カスタマージャーニー全体の可視化
   • アトリビューション分析
   • 実際の貢献度を定量化
   ```

3. **UTMパラメータの統一**
   ```
   設定例：
   • utm_source=moodmarkgift
   • utm_medium=article
   • utm_campaign=birthday_gift
   • utm_content=article_id
   ```

**🟡 優先度：中**

4. **ECサイト人気商品のSEOメディアでの特集**
   - ECサイトのベストセラーを記事化
   - 季節・イベントに合わせた特集記事
   - 商品レビュー記事の作成

5. **リターゲティング広告の活用**
   - SEOメディア訪問者にECサイトの広告配信
   - ECサイト訪問者にSEOメディアの記事配信
   - 両サイト訪問者の購買促進
